# GitHub Webhooks to Discord Notifier using Netlify Functions
A serverless application built with Netlify Functions that listens to GitHub webhooks and sends real-time repository notifications to Discord.

## Requirements
- Node.js
- Netlify Account
- Netlify CLI (authenticated)
- GitHub repository (your choice)

## Project setup and Deploy to production
1. Run the following command to install project dependencies:
```bash
npm install
```
2. Make a copy of `.env.example` and rename the file with `.env`.
3. To set the `DISCORD_WEBHOOK_URL` create a server or use an existing one, then go to `Server setting` > `Integrations` > `Webhooks` > `New webhook`.
4. Click on the Webhook and click on the `Copy webhook URL` and set it in the `DISCORD_WEBHOOK_URL` variable environment.
5. (Optional) You can test serverless function on local by running the following command:
```bash
netlify dev
```
6. Run the following command to deploy to production on Netlify and save the deployed url:
```bash
netlify deploy --prod
```
7. Go to a GitHub repository that you want to receive notifications about.
8. Go to `Settings` > `Webhooks` and click on `Add webhook`.
9. Set the Webhook with the following configuration:
  - `Payload URL`: Url generated by Netlify CLI on deploy to production `{{PUBLIC_NETLIFY_URL}}/.netlify/functions/github-discord`.
  - `Content type`: application/json
  - `Secret`: No needed for this project. Lightweight implementation without signature verification (educational purpose)
  - `SSL verification`: Enable SSL verification
  - `Which events would you like to trigger this webhook?`: Let me select individual events (select `Stars` and `Issues`)

> [!WARNING]
> This project only supports `Stars` and `Issues` GitHub events, other events will be ignored and logged as "Unknown event".
> This project does NOT verify the GitHub webhook signature. It is intended for learning purposes only and should not be used in production without signature validation.

> [!TIP]
> Now every event related to issues and stars in the repository will be logged in the discord server.
